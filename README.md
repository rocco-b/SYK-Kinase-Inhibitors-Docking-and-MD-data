## Repository content
The repository contains all input and output files used for molecular docking.
* In the case of single docked molecules, the output file is named *docked.pdb*.
* In the case of virtual screening, the output files are contained in the “outputs” subfolder and named *molecule_1, molecule_2, ..., molecule_N*.
* The *log.txt* files contain the textual output of GNINA with the *CNN score* and *CNNaffinity* values expressed.
